# Kwork Лента

Приложение читает **открытую ленту заказов** с биржи Kwork и показывает
её в удобном виде: фильтры по бюджету, поиск по ключевым словам,
категории, избранное. Тап на заказ → открывается полная страница на
Kwork, где ты логинишься и откликаешься.

## Что это умеет

- 🔥 Лента открытых заказов Kwork (биржа) — парсится при открытии
- 🔎 Фильтр по словам: "бот", "python", "сайт" и т.д.
- 💰 Фильтр по бюджету (от/до)
- 📂 Категории: разработка, боты, дизайн, тексты
- ⭐ Избранное — сохраняй заказы локально
- 🔗 Открытие заказа на Kwork в один тап

## Чего НЕ умеет (и почему)

У Kwork нет публичного API. Приложение работает только с тем, что
видно **без логина** — открытая биржа заказов. Поэтому:

- ❌ Не читает твои личные сообщения и заказы (они за логином)
- ❌ Не отправляет сообщения от твоего имени
- ❌ Не шлёт пуш-уведомления (для этого нужен сервер 24/7)

Чтобы откликнуться или написать заказчику — тап на заказ открывает
его на Kwork, и дальше ты работаешь как обычно в своём аккаунте.

## ВАЖНО: работает только в APK

Парсинг ленты обходит защиту браузера (CORS) через нативный HTTP
Capacitor. В обычном браузере (PWA) это **не работает** — будет ошибка.
**Используй собранный APK.**

## Сборка APK

Запушь в GitHub → APK соберётся автоматически (Actions → Artifacts).

Или локально:
```bash
npm install
npx cap add android
npx cap sync android
cd android && ./gradlew assembleDebug
```

## Если заказы не загружаются

Kwork мог изменить структуру страницы. В приложении на экране ошибки
есть раздел "Технические детали" — раскрой его, скопируй текст и
пришли разработчику, парсер поправят под новую структуру.
